
**ATM Program**

*Overview*

This is part A of the c++ Assignment 2, users can:

Create a new bank account with a user ID and password.
Log in to an account.
Quit the program.

*In Part B, I will add functions to deposit, withdraw, and check the balance.*

*How It Works*
Create Account: The user enters a user ID and password to create an account.
Log In: The user can log in using the same ID and password.
Quit: Exits the program.

*File Structure*
BankAccount.h - Declares the BankAccount class, including user ID, password, and balance.
BankAccount.cpp - Defines the functions for the BankAccount class.
main.cpp - Manages the main menu and user interaction.
